# 包子智能代理 v2.4

基于人脑记忆模型的AI助手

---

## 核心架构

### 记忆系统 v4.0（基于人脑神经科学）

```
用户消息 → 感觉记忆 → 工作记忆 → 长时记忆
```

| 记忆类型 | 存储位置 | 说明 |
|---------|---------|------|
| 感觉记忆 | 内存 | 瞬时，不存储 |
| 工作记忆 | working-buffer.md | session临时 |
| 情景记忆 | diary/ | 具体事件 |
| 语义记忆 | USER.md, LEARNINGS.md | 偏好、知识 |
| 程序记忆 | SKILL.md | 技能规则 |
| 重要记忆 | important/ | 标记重要 |

---

## 联动机制

### 每次响应流程

```
用户消息 
    ↓
分析内容 → 调用 smart-link.js
    ↓
自动分类写入正确文件
    ↓
更新 index.json
```

### 分类规则

| 内容类型 | 目标文件 | 关键词 |
|---------|---------|--------|
| 偏好/设置 | USER.md | 偏好、设置、参数 |
| 纠正/否定 | LEARNINGS.md | 不是、其实、纠正 |
| 错误/失败 | ERRORS.md | 错误、失败 |
| 任务/待办 | SESSION-STATE.md | 任务、待办 |
| 重要信息 | important/ | 重要、关键 |
| 一般记录 | diary/ | 其他 |

---

## 心跳机制（15分钟一次）

### 三大功能

```
① 状态检查 → 系统是否正常
② 联动整理 → 整理记录的文件  
③ 主动记忆 → 想起重要的事提醒你
```
